# **App Name**: MediAssistant AI

## Core Features:

- User Authentication: Secure signup, login, and password recovery for user accounts.
- AI-Powered Chat Interface: A conversational chatbot that remembers context and intent, providing information about medicines, diseases, dosages, and usage periods based on user input. The tool should consider its understanding of the conversation context to decide how to best incorporate new information.
- Diseases Library: A searchable repository of diseases with detailed information. Displays symptoms, causes, control measures, types, examples, emergency contacts and doctor consultation advice.
- Medicine Scheduling: Functionality to set medicine schedules with date, time, frequency (daily/weekly), and custom sound/ringtone alarms.
- Medicine Information Search: AI-driven search for medicine information across medical websites. Responses are presented with clickable links to the source websites.
- Symptom Checker: Users can input health symptoms, and the AI will provide potential explanations.
- Settings Page: Page where the user manages his general settings such as notifications or theming.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to convey trust and tranquility.
- Background color: Very light blue (#F0F8FF), near-white, to maintain a clean and calming feel.
- Accent color: Pale violet (#BDB2FF) to highlight important information.
- Body font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality, for the body of text.
- Headline font: 'Playfair', a modern sans-serif with high contrast thin-thick lines, for a elegant, high-end feel for the headers.
- Use clear and easily recognizable icons, related to medical themes, in a consistent style.
- Clean and intuitive layout with clear sectioning to improve user experience.
- Subtle transition animations to acknowledge user interactions.
